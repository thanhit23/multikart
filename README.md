# Project Admin Multikart

## Quick Start

Clone repo
```
git clone https://github.com/thanhit23/multikart.git
cd multikart
npm install
```

Set the environment variables:

```
cp .env.example .env
```

# Project Structure

```
src\
  |-component
  |-container
  |-helper
  |-hook
  |-layouts
  |-middleware
  |-routes
  |-service
  |-translate
  |-utils
```
